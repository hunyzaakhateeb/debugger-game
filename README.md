# Error Finder — Debug the Code

**Tagline:** Find the bug before the timer expires!

## Short Description
Error Finder is a browser-based debugging mini-game where players read short buggy code snippets and choose the type of bug. Players have 2 lives and faster correct answers yield higher points. The game saves top scores in localStorage.

## How to Run
1. Ensure files are placed in a folder:

2. Open `index.html` in a modern web browser (Chrome/Edge/Firefox). No server required — runs locally offline.

## Game Rules / Instructions
- You will be shown a short code snippet with a single bug each round.
- Select the bug type from 4 options.
- Controls:
- Click an option or press `1`–`4` to select.
- Press `Enter` or click **Submit** to submit.
- **Skip** costs 1 life.
- 2 wrong answers (or 0 lives) = Game Over.
- Scoring: Faster answers yield more points (0–5 points per correct answer).
- Top scores are saved in your browser's localStorage (Scoreboard).

## Notes
- No internet access required. Sounds are created in-browser using WebAudio.
